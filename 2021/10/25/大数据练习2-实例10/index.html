实例10 使用IDEA编写WordCount程序

1、新建Scala Object

在practise3包鼠标右键，选择New->Scala Class命令，用于新建Scala文件

在弹出的对话框中，选择Kind为Object

在刚刚新建的文件中添加代码

def main(args: Array[String]) : Unit = {

    val conf = new SparkConf()
      .setMaster("spark:192.168.222.128:9000")
      .setAppName("WC")
    val sc = new SparkContext(conf)
    sc.textFile("hdfs://192.168.222.128:9000/user/hadoop/word2.txt")
      .flatMap(_.split(" "))
      .map((_, 1))
      .reduceByKey(_ + _, 1)
      .sortBy(_._2, false)
      .saveAsTextFile("hdfs://192.168.222.128:9000/output")
    sc.stop()
  }

运行代码，去HDFS中查找output文件
